## 🎮 How to Play  

1. **Arrow Keys** → Move the snake.  
2. **Eat the red dot** → Grow and increase your score.  
3. **Avoid walls and yourself** → Or it's **Game Over**!  
4. **Press SPACE / ENTER** → Start a new game.
